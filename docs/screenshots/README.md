# MoodMitra AI – Design Screenshots

High-fidelity mockups of every screen state and UI component, generated to verify the glassmorphic dark-theme design against the specification.

## Screens

| # | File | State |
|---|------|-------|
| 1 | `01_empty_stub_mode.png` | Empty chat, stub mode (model not loaded) |
| 2 | `02_active_conversation.png` | Active conversation with emotion-aware chips |
| 3 | `03_camera_emotion.png` | Camera panel open + emotion indicator + activity captions |
| 4 | `04_ai_generating.png` | AI streaming response (cursor + generating banner) |
| 5 | `05_voice_listening.png` | Voice input active (rose mic, waveform overlay) |
| 6 | `06_component_gallery.png` | Component gallery (all chips, bubbles, badges, buttons) |
